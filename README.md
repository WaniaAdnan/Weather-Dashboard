## 🌍 About the Project

**Weather Dashboard** is a web-based application that allows users to search for any city and instantly view **real-time weather data** along with a **5-day forecast**. It fetches live data using the **OpenWeather API** and presents it through a clean, responsive interface.

🔗 **Live Demo:** [waniaadnan.github.io/Weather-Dashboard](https://waniaadnan.github.io/Weather-Dashboard/)

---

## ✨ Features

- 🔍 Search any city worldwide
- 🌡️ Real-time data: temperature, humidity, wind speed & conditions
- 📅 5-day weather forecast
- 🕘 Search history stored for previously searched cities
- 📱 Dynamic and fully responsive UI

## 🚀 How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/WaniaAdnan/Weather-Dashboard.git
   ```
2. Get a free API key from [openweathermap.org](https://openweathermap.org/api)
3. Add your API key in `script.js` where indicated
4. Open `index.html` in your browser
